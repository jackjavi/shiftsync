<h1 align="center">ShiftSync Server</h1>

This is the server application for ShiftSync, the leading staffing marketplace on Mars. Built with NestJS and Prisma, it provides a robust API for managing workers, workplaces, and shifts.

## Technologies

- [TypeScript](https://www.typescriptlang.org/)
- [NestJS](https://docs.nestjs.com/)
- [Prisma](https://www.prisma.io/docs/concepts/components/prisma-client)
